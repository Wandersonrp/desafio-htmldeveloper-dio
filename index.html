Objetivo do desafio

(voltar)

O objetivo do presente desafio é colocar em prática o aprendizado adquirido no primeiro módulo do curso de formação HTML Developer, da DIO.

Para isso, a metodologia de realização do desafio será explicar com suas palavras as diversas tags apresentadas no primeiro módulo do curso de formação.

Conteúdo do desafio

(voltar)

O conteúdo do desafio será a explicação das tags ensinadas no módulo 01 do curso de formação, bem como sua execução prática ao elaborar a página HTML utilizando as tags.

Tags de texto

Parágrafo

No HTML, um parágrafo é representado pela tag <p> </p> e é utilizado para a inserção de textos.

Itálico

O itálico, no HTML, é representado pela tag <i> </i>.

Negrito

Para utilizar negrito em um texto ou em parte dele, é utilizada a tag <b> </b>.

Sublinhado

Para utilizar sublinhado em um texto ou em parte dele, é utilizada a tag <u> </u>.

Marca-texto

Para utilizar marca-texto em um texto ou em parte dele, é utilizada a tag <mark> </mark>.

Citação

Para fazer uma citação no HTML, a tag utilizada é <blockquote> </blockquote>.
